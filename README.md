# Mine-Plus-Plus
Minecraft-style game created using C++ and OpenGL

## Done
- Chunk render
- Simple world generation
- Build/destroy mechanic
- Lighting 

## Controls
- WASD, mouse, lshift, lctrl - camera controls
- Space - increase camera speed
- P - on/off vsync
- O - save map
- L - load saved map
- F3 - debug info
